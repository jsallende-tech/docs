# Application Release Note

Each time when a new application or a new version of an existing app is introduced, Partner is obliged to deliver a filled-out Application Release Note (previously called Product Delivery Note) document.

## Template

<Card title="Download ARN Template" icon="file-word" href="/assets/smartpos/files/application-release-note-template.docx">
  Download the Application Release Note template (Word document)
</Card>

The ARN template can also be found as an attachment to the Business Application agreement signed with WL for app delivery.

## Submission

Once filled out, the ARN document shall be **sent to**:

<Card title="Android App Partner" icon="envelope" href="mailto:Android.AppPartner@worldline.com">
  Android.AppPartner@worldline.com
</Card>
