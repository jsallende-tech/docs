WPI 2.1 builds upon version 2.0 with additional features and improvements for SmartPOS integration.

## Migration from 2.0

If you're currently using WPI 2.0, migration to 2.1 is straightforward:

1. Update the `WPI_VERSION` value in your implementation
2. Review new optional fields that may enhance your integration
3. Test with the updated payment solution

<Note>
WPI 2.1 is backward compatible with 2.0. Existing implementations will continue to work without modification.
</Note>
