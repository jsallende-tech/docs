# How to Set Up Your Terminal

## Downloading Terminal API

<Note>
If Terminal API application is visible in the launcher, skip this section.

For S1U2: You need to tap on the Software "MENU" button that appears after a reboot until the TID + password are displayed.
</Note>

<Steps>
  <Step title="Access the Launcher">
    Access the launcher on your payment terminal by pressing the circle at the bottom of the screen. If the circle isn't visible, swipe up from the bottom of the screen to reveal the navigation menu.
  </Step>
  <Step title="Open App Drawer">
    Swipe up from the bottom of the screen to access the App Drawer. You will see a list of all installed applications.
  </Step>
  <Step title="Open SmartPOS Store">
    Tap on the "SmartPOS Store" app. The icon may have changed, but the app name remains the same.
  </Step>
  <Step title="Download Terminal API">
    In the "All" tab of the store, click on Terminal API cloud icon for it to be downloaded into the terminal.
  </Step>
</Steps>

---

## Terminal API Initial Configuration

To ensure proper functionality of our Terminal API on your payment terminal, please follow these configuration steps:

<Steps>
  <Step title="Access the Launcher">
    Access the launcher on your payment terminal by pressing the circle at the bottom of the screen. If the circle isn't visible, swipe up from the bottom of the screen to reveal the navigation menu.
  </Step>
  <Step title="Open App Drawer">
    Swipe up from the bottom of the screen to access the App Drawer. You will see a list of all installed applications.
  </Step>
  <Step title="Open Terminal API">
    Tap on the "Terminal API" app. The icon may have changed, but the app name remains the same.
  </Step>
  <Step title="Enter Credentials">
    Enter the data provided to you (UMID and UTID). Double-check this information, as any errors will prevent server communication.
  </Step>
  <Step title="Select Environment">
    Select the appropriate environment to connect to:
    - **PRODUCTION** for live operations
    - **INTEGRATION** for testing with test terminals and cards
  </Step>
  <Step title="Activate Terminal API">
    Activate the Terminal API by tapping the slider next to it. Ensure the cloud icon next to Terminal API turns green.
  </Step>
</Steps>

<Info>
You are done! Now you can communicate Cloud to Cloud with your cash register thanks to Terminal API.
</Info>

## Setting Up Terminal API as the Default Application

After every reboot, the terminal goes back to its default state. This may affect the general user experience. One possibility is to setup Terminal API as the default application.

<Steps>
  <Step title="Access the Launcher">
    Access the launcher on your payment terminal by pressing the circle at the bottom of the screen.
  </Step>
  <Step title="Open App Drawer">
    Swipe up from the bottom of the screen to access the App Drawer.
  </Step>
  <Step title="Open Settings API">
    Tap on the "Settings API" app. The icon may have changed, but the app name remains the same.
  </Step>
  <Step title="Select Default Application">
    Tap into the default application selection. You will be offered a choice of apps.
  </Step>
  <Step title="Choose Terminal API">
    Select Terminal API.
  </Step>
  <Step title="Complete Setup">
    Go back, once you exit this App, Terminal API APP will pop up automatically after every reboot.
  </Step>
</Steps>

## Initialization and Configuration

The initialization and configuration are run periodically, but if you want to ensure that you have the latest settings if something is pending:

<Steps>
  <Step title="Access the Launcher">
    Access the launcher on your payment terminal.
  </Step>
  <Step title="Open Setup Menu">
    Click on the Setup menu.
  </Step>
  <Step title="Enter Password">
    Enter the password.
  </Step>
  <Step title="Run Configuration">
    Click on "Configuration" and afterwards on "Initialisation".
  </Step>
</Steps>
