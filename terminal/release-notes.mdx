## Versioning

Terminal API follows semantic versioning aligned with Nexo Retailer versions:

- **Major version** (5.x) corresponds to Nexo Retailer major version
- **Minor version** (x.1) corresponds to Nexo Retailer minor version
- **WL version** (-WL1.0.0) indicates Worldline-specific extensions

<Note>
Always review release notes before upgrading to understand any breaking changes or required migrations.
</Note>
