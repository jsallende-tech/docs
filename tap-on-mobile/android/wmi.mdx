# Worldline Mobile Interface (WMI)

WMI provides an alternative integration approach for Tap on Mobile on Android devices.

## Overview

WMI is designed specifically for mobile payment acceptance scenarios with optimized APIs for smartphone integration.

## Features

- Mobile-optimized APIs
- Simplified integration flow
- Full contactless payment support
