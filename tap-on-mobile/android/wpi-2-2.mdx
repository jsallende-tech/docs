# WPI 2.2 for Tap on Mobile

The Worldline Payment Interface (WPI) 2.2 extends the standard WPI interface for Tap on Mobile on Android devices.

## Overview

WPI 2.2 builds on the existing WPI interface used in SmartPOS, adapted for smartphone-based payment acceptance.

## Key Features

- Contactless card acceptance via NFC
- Intent-based integration
- Consistent API with SmartPOS WPI
