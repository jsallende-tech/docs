# Tap on Mobile Overview

Transform smartphones into secure payment acceptance devices. Tap on Mobile allows merchants to accept contactless payments directly on their iOS and Android devices.

## Key Features

<CardGroup cols={3}>
  <Card title="Contactless Payments" icon="wifi">
    Accept NFC-based contactless card payments
  </Card>
  <Card title="iOS & Android" icon="mobile">
    Available on both major mobile platforms
  </Card>
  <Card title="Secure" icon="shield">
    PCI-compliant payment processing
  </Card>
</CardGroup>
